# Standalone Eval Design

This folder contains the source audit, literature survey, and design plan for an original handwritten evaluation function for `gen9randombattle`.

Files:

- `fp_eval_audit.md` — source-level audit of Foul Play / poke-engine's current gen9 eval and search-adjacent machinery.
- `eval_literature_survey.md` — cited sources for handcrafted eval design, imperfect-information search, tuning, and Pokemon-specific lessons.
- `standalone_eval_plan.md` — proposed original eval architecture, feature groups, tuning plan, and gates.

Purpose: make a bot whose credited search infrastructure may still come from Foul Play / poke-engine, but whose evaluation function and belief-aware state terms are original and independently documented.
